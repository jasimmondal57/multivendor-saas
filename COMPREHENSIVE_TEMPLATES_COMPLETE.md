## ðŸš€ Running the Seeders

To populate all templates in the database:

```bash

```

## ðŸ”§ Usage Example

### **Email Template**
```php
use App\Models\EmailTemplate;

$template = EmailTemplate::getByCode('customer_order_confirmation');
$emailContent = $template->render([
    'customer_name' => 'John Doe',
    'order_number' => 'ORD-12345',
    'order_total' => '2,500',
    'order_url' => 'https://example.com/orders/12345',
    'site_name' => 'MyStore'
]);
```

### **WhatsApp Template**
```php
use App\Models\WhatsAppTemplate;

$template = WhatsAppTemplate::where('code', 'customer_order_shipped')->first();
$message = $template->formatMessage([
    'John Doe',
    'ORD-12345',
    'ABC123456',
    'https://track.example.com/ABC123456'
]);
```
